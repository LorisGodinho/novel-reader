```json
  "seletores": {

    "botao_proximo": "SELETOR_CSS_AQUI",
    "lista_capitulos": "SELETOR_CSS_AQUI"
  }
}
```

### 3. Como Encontrar Seletores

1. Abra o capítulo no navegador
2. Pressione `F12` para abrir DevTools
3. Clique no ícone de seleção (Ctrl+Shift+C)
4. Clique no elemento desejado
5. No painel Elements, clique direito > Copy > Copy selector
